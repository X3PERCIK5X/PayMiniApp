# PayMiniApp

Мини-приложение оплаты подписки (3000 ₽) и Telegram-бот для запуска мини-аппа.

## Файлы

- `Оплата мини апп.html` — фронтенд мини-аппа
- `бот/оплата бот.js` — код Telegram-бота

## Настройка бота

Создайте `.env` рядом с `бот/оплата бот.js`:

```env
BOT_TOKEN=ваш_токен_бота
WEBAPP_URL=https://ваш-домен/Оплата%20мини%20апп.html
TINKOFF_PAYMENT_URL=https://ваша-ссылка-tinkoff
```

Установите зависимости и запустите:

```bash
npm i node-telegram-bot-api dotenv
node "бот/оплата бот.js"
```
